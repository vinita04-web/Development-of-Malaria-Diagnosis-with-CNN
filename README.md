Project Title

Development of Malaria Diagnosis with Convolutional Neural Network Architectures

Objective

To classify microscopic blood cell images into:

Parasitized
Uninfected

using the proposed CNN model from the research paper.

Dataset

Malaria Cell Images Dataset

https://www.kaggle.com/datasets/iarunava/cell-images-for-detecting-malaria

Technologies Used
Python
TensorFlow/Keras
OpenCV
NumPy
Matplotlib
Scikit-learn
Jupyter Notebook
Workflow
Load dataset
Resize images to 50×50×3
Normalize images
Split into train, validation, and test sets
Build the custom CNN model
Train for 50 epochs
Evaluate the model
Generate classification report and confusion matrix
Predict on random test images
Save the trained model
